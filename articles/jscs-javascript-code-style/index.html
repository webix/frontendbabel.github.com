JSCS: JavaScript Code Style

Written by Marat Dulin on 21st April 2014; translated by Varya Stepanova on 15th July 2014

I began to develop JSCS a year ago and did not expect it to become a robust tool, which is unique and used by such famous teams like jQuery, Bootstrap and Angular. By now, at the moment of publishing this article, my project`s repository has gained over 1400 stars and I am vary glad to realize how many people were enabled to make their work more comfortable by my product.

This project originated in my own pain.
